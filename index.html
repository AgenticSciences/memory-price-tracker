9.Methodology and Limitations

9.1 Data Collection

Current prices are collected directly from DRAMeXchange (TrendForce) via automated web scraping. The process extracts session averages, daily/weekly ranges, and percentage changes for each product category.

9.2 Historical Data

Historical price points are verified from The Memory Guy's published analysis, which sources data from InSpectrum. These represent actual recorded spot prices per gigabyte for branded DRAM products. Interpolated or estimated historical data is not included in this index.

9.3 Limitations

  • Historical data points are sporadic; continuous time-series data requires paid subscriptions to market research services
  • Spot prices may not reflect contract prices paid by large-volume buyers
  • Component prices do not include manufacturing, testing, or distribution costs for finished modules
  • Regional price variations are not reflected in this index

9.4 Use as Perpetual Futures Benchmark

This index is designed to serve as an underlying reference for cash-settled perpetual futures contracts on memory products. The spot prices of DRAM and NAND Flash components represent the raw input costs for memory manufacturers and datacenter operators. For trading purposes, verify all data with primary sources and understand the distinction between spot and contract pricing.
